Proposito del sitio

El propósito de este sitio es enseñar desde los fundamentos hasta el uso de HTML como herramienta para la creación de sitios web basandonos en las ventajas de HTML como lo son: Estructuras amenas, no se requieren conocimientos avanzados simpre y cuando contemos con editores web, compatibilidad con todos los navegadores.

Fundamentos de HTML

Definición: HTML (Hyper Text Markup Language) es un lenguaje para el desarrollo de paginas web. Este lenguaje nos permite escribir texto de manera estructurada, compuesto por etiquetas las cuales marcan el inicio y final de cada uno de los elementos. Para mostrar las paginas web creadas es necesario el uso de navegadores web los cuales interpretan el código HTML. Catalunya (26 de septiembre de 2022) Elementos: * Etiquuetas * Pares de etiquetas * Etiquetas con atributos * Etiquetas anidadas

Interacción Humano Ordenador

Según Brenton (Julio 22 de 2019) La interacción Humano-Computadora es una disciplina que se encarga de estudiar cómo los seres humanos interactúan con los ordenadores y el punto en el que un ordenador es capáz de interactuar con las personas. La interacción Humano-Computadora tiene tres elementos claves: las personas, las computadoras y la interacción entre ellos. La interacción tiene como propósito; que sean faciles de usar y aprender, confiable y seguro y funcional, es decir que sea de acuerdo a las necesidades de las personas.
